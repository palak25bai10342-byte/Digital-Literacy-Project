# 🌐 Digital Literacy Project

Hi! I'm Palak Verma 👋  
This project is part of my CSE0001 Digital Literacy course at VIT Bhopal.

The idea behind this project was to explore what digital literacy actually means in real life — not just theory, but how we use digital tools, communicate professionally, and stay safe online.



## 📌 What This Project Includes

This repository is divided into 5 tasks, each focusing on a different aspect of digital literacy:



## 🎨 Task 1 – Digital Literacy Infographic  
I created an infographic using Canva to explain key concepts like digital literacy, useful tools, safe internet practices, and email etiquette in a simple and visual way.



## 🌐 Task 2 – Digital Portfolio  
I set up and updated my profiles on:
- GitHub  
- LinkedIn  
- Kaggle  

These platforms will help me build my professional presence and showcase my work over time.



## 💻 Task 3 – Platforms Exploration  
- Solved a beginner coding problem on HackerRank  
- Created a Digital Literacy Awareness Quiz using Google Forms

## 📧 Task 4 – Email Etiquette  
I wrote professional emails and created a checklist for responsible social media usage, focusing on clear and respectful communication.



## 🔐 Task 5 – Cybercrime Awareness  
I explored a case study on Cyberbullying and created a prevention checklist to understand how to stay safe in the digital world.



## 🧠 What I Learned

This project made me realize that digital literacy is not just about using technology, but about using it responsibly and effectively.  
I learned how to present myself online, communicate professionally, and be more aware of online risks.



## 🚀 Final Thoughts

Overall, this project helped me become more confident with digital tools and understand their importance in both academic and real-life situations.
